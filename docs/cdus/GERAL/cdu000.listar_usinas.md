# **CDU000 – Listar Usinas**

* **Ator principal**: Empresa, Gerente
* **Atores secundários**: Sistema
* **Resumo**: Permite a empresa/gerente listar todas as usinas cadastradas da empresa, com filtros por status, localizacao e capacidade instalada.
* **Pré-condição**:

	* A empresa/gerente deve estar autenticado no sistema.
	* A empresa/gerente deve estar ativo.

* **Pós-condição**:

	* A listagem de usinas e exibida conforme os filtros aplicados.

## **Fluxo Principal**

|               **Ações do Ator**               |                              **Ações do Sistema**                              |
| :-------------------------------------------: | :----------------------------------------------------------------------------: |
| 1 - Acessa a opção “Listar Usinas”.            |                                                                                |
