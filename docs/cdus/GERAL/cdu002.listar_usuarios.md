# **CDU002 – Listar Usuários**

* **Ator principal**: Empresa, Gerente
* **Atores secundários**: Sistema
* **Resumo**: Permite a empresa/gerente listar usuários operacionais da plataforma, apresentando username, email, status e tipo.
* **Pré-condição**:

  * A empresa/gerente deve estar autenticado no sistema.
  * A empresa/gerente deve estar ativo.

* **Pós-condição**:

  * A listagem de usuários é exibida conforme os filtros aplicados.

---
## **Mapeamento CDU -> Requisitos Funcionais**

| **Requisito** | **Descrição**               |
| ------------- | --------------------------- |
| RF05          | Listagem de usuários        |

---
## **Mapeamento CDU -> Regras de Negócio**

| **Código** | **Descrição**                                                                                                                                                |
| :--------: | :----------------------------------------------------------------------------------------------------------------------------------------------------------- |
|  **RN02**  | Apenas empresas podem gerenciar usuários do tipo gerente (cadastrar, atualizar, listar, ativar/desativar e controlar permissões e perfil).                   |
|  **RN05**  | O gerente pode cadastrar, atualizar, listar, ativar ou desativar apenas usuários vinculados à mesma empresa à qual ele pertence.                              |

---
## **Fluxo Principal**

|               **Ações do Ator**               |                              **Ações do Sistema**                              |
| :-------------------------------------------: | :----------------------------------------------------------------------------: |
| 1 - Acessa a opção “Listar Usuários”.          |                                                                                |
|                                               | 2 - Recupera usuários vinculados à empresa do usuário.                         |
|                                               | 3 - Exibe a listagem com filtros por nome, tipo e status.                       |
| 4 - Visualiza a listagem de usuários.          |                                                                                |
