# **CDU000 – Atualizar Empresas**

* **Ator principal**: Empresa
* **Atores secundários**: Sistema
* **Resumo**: Permite a empresa atualizar seus dados cadastrais e de contato na plataforma, mantendo o histórico das alterações.
* **Pré-condição**:

	* A empresa deve estar autenticada no sistema.
	* A empresa deve estar ativa.

* **Pós-condição**:

	* Dados da empresa são atualizados e o histórico das alterações é preservado.

## **Fluxo Principal**

|               **Ações do Ator**               |                              **Ações do Sistema**                               |
| :-------------------------------------------: | :------------------------------------------------------------------------------: |
| 1 - Acessa a opção “Atualizar Empresa”.        |                                                                                  |
|                                               | 2 - Exibe os dados cadastrais atuais da empresa.                                 |
| 3 - Edita os campos desejados e confirma.      |                                                                                  |
|                                               | 4 - Valida dados obrigatórios e consistência dos campos.                         |
|                                               | 5 - Salva as alterações e registra o histórico das mudanças.                     |
| 6 - Visualiza a confirmação da atualização.   |                                                                                  |
