# **CDU000 – Atualizar Vendedores**

* **Ator principal**: Empresa, Gerente
* **Atores secundários**: Sistema
* **Resumo**: Permite a empresa/gerente atualizar os dados cadastrais de vendedores cadastrados para controle comercial e comissionamento.
* **Pré-condição**:

	* A empresa/gerente deve estar autenticado no sistema.
	* A empresa/gerente deve estar ativo.

* **Pós-condição**:

	* Dados do vendedor sao atualizados e o historico das alteracoes e preservado.

---
## **Mapeamento CDU -> Requisitos Funcionais**

| **Requisito** | **Descrição**            |

---
## **Mapeamento CDU -> Regras de Negócio**

| **Código** | **Descrição**                                                                                                  |
| :--------: | :------------------------------------------------------------------------------------------------------------- |
|  **RN09**  | Vendedores cadastradas não possuem acesso direto ao sistema, suas informações ficam salvas, porém não possuem credencias de acesso. |

---
## **Fluxo Principal**

|                **Ações do Ator**                 |                              **Ações do Sistema**                              |
| :----------------------------------------------: | :----------------------------------------------------------------------------: |
| 1 - Acessa a opção “Listar Vendedores”.          |                                                                                |
|                                                  | 2 - Exibe vendedores vinculados a empresa.                                     |
| 3 - Seleciona um vendedor para editar.           |                                                                                |
|                                                  | 4 - Exibe formulario de edicao com dados atuais.                               |
| 5 - Atualiza os dados e confirma.                |                                                                                |
|                                                  | 6 - Valida campos obrigatorios e consistencia dos dados.                       |
|                                                  | 7 - Salva as alteracoes e registra o historico.                                |
| 8 - Visualiza os dados atualizados do vendedor.  |                                                                                |
