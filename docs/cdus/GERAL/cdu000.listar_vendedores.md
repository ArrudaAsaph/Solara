# **CDU000 – Listar Vendedores**

* **Ator principal**: Empresa, Gerente
* **Atores secundários**: Sistema
* **Resumo**: Permite a empresa/gerente listar vendedores cadastrados para controle comercial e comissionamento.
* **Pré-condição**:

	* A empresa/gerente deve estar autenticado no sistema.
	* A empresa/gerente deve estar ativo.

* **Pós-condição**:

	* A listagem de vendedores e exibida conforme os filtros aplicados.

---
## **Mapeamento CDU -> Requisitos Funcionais**

| **Requisito** | **Descrição**            |
| ------------- | ------------------------ |
| RF06          | Cadastro de Vendedores   |

---
## **Mapeamento CDU -> Regras de Negócio**

| **Código** | **Descrição**                                                                                                  |
| :--------: | :------------------------------------------------------------------------------------------------------------- |
|  **RN09**  | Vendedores cadastradas não possuem acesso direto ao sistema, suas informações ficam salvas, porém não possuem credencias de acesso. |

---
## **Fluxo Principal**

|               **Ações do Ator**               |                         **Ações do Sistema**                         |
| :-------------------------------------------: | :------------------------------------------------------------------: |
| 1 - Acessa a opção “Listar Vendedores”.        |                                                                      |
|                                               | 2 - Recupera vendedores vinculados a empresa do usuario.             |
|                                               | 3 - Exibe listagem com filtros por nome, status e periodo.           |
| 4 - Visualiza a listagem de vendedores.        |                                                                      |

---
## **Fluxos Alternativos**

### **Fluxo Alternativo I – [Filtrar listagem de vendedores]**

|               **Ações do Ator**               |                        **Ações do Sistema**                       |
| :-------------------------------------------: | :---------------------------------------------------------------: |
| 1.1 - Informa filtros desejados.               |                                                                 |
|                                               | 1.2 - Aplica filtros e atualiza a listagem.                        |
| 1.3 - Visualiza a listagem filtrada.           |                                                                 |

---
## **Fluxos de Exceção**
