# CDU007 – Listar Vendedores

> **Arquivo:** `cdu007_listar_vendedores.md`
> **Descrição:** Caso de uso referente à listagem de vendedores cadastrados na empresa para fins de controle comercial e comissionamento. Vendedores não são usuários do sistema — não possuem credenciais de acesso ([RN09](../../regras_de_negocio/regras_de_negocio.md#rn09)).

---

## Identificação

| **Campo** | **Valor** |
| :-------- | :-------- |
| **Código** | CDU007 |
| **Ator Principal** | Empresa / Gerente |
| **Atores Secundários** | Sistema |
| **Requisito Atendido** | [RF21](../../requisitos/requisitos_funcionais.md#rf21) |

---

## Resumo

Permite à Empresa ou ao Gerente visualizar todos os vendedores cadastrados e vinculados à empresa, podendo aplicar filtros por nome, status e período. Por não serem usuários do sistema, vendedores não aparecem na listagem de usuários (CDU002).

---

## Pré-condições

- O ator deve estar autenticado no sistema.
- O ator deve estar ativo e vinculado a uma empresa ativa.

---

## Pós-condições

- A listagem de vendedores vinculados à empresa é exibida conforme os filtros aplicados.
- Nenhum dado é alterado no sistema.

---

## Mapeamento: CDU → Regras de Negócio

| **Código** | **Descrição** |
| :--------: | :------------ |
| [**RN23**](../../regras_de_negocio/regras_de_negocio.md#rn23) | Empresa e gerente podem listar vendedores apenas da mesma empresa à qual estão vinculados. |
| [**RN09**](../../regras_de_negocio/regras_de_negocio.md#rn09) | Vendedores cadastrados não possuem acesso direto ao sistema. Suas informações ficam salvas para fins de controle financeiro e comissionamento, porém não possuem credenciais de acesso. |

---

## Fluxo Principal

| **Passo** | **Ações do Ator** | **Ações do Sistema** |
| :-------: | :---------------- | :------------------- |
| 1 | Acessa a opção "Listar Vendedores". | |
| 2 | | Recupera todos os vendedores vinculados à empresa do ator. |
| 3 | | Exibe a listagem com as informações: nome, status e período de cadastro. |
| 4 | Aplica filtros por nome, status ou período (opcional). | |
| 5 | | Atualiza a listagem conforme os filtros aplicados. |
| 6 | Visualiza os resultados da listagem. | |
