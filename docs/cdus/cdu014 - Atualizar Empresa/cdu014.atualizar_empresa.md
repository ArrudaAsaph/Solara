# CDU014 – Atualizar Empresa

> **Arquivo:** `cdu014_atualizar_empresa.md`
> **Descrição:** Caso de uso referente à atualização dos dados cadastrais e de contato da própria empresa no sistema.

---

## Identificação

| **Campo** | **Valor** |
| :-------- | :-------- |
| **Código** | CDU014 |
| **Ator Principal** | Empresa |
| **Atores Secundários** | Sistema |
| **Requisito Atendido** | RF23 |

---

## Resumo

Permite que a empresa atualize seus próprios dados cadastrais e de contato, mantendo histórico das alterações. Não é permitido atualizar dados de outra empresa.

---

## Pré-condições

- A empresa deve estar autenticada no sistema.
- A empresa deve estar ativa.

---

## Pós-condições

- Os dados da empresa são atualizados com sucesso.
- O histórico das alterações é preservado.

---

## Mapeamento: CDU → Requisitos Funcionais

| **Requisito** | **Descrição** |
| :------------ | :------------ |
| **RF23** | A empresa pode atualizar os próprios dados cadastrais e de contato, mantendo histórico das alterações. |

---

## Mapeamento: CDU → Regras de Negócio

| **Código** | **Descrição** |
| :--------: | :------------ |
| **RN26** | A empresa pode atualizar apenas os próprios dados cadastrais e de contato; não pode atualizar dados de outra empresa. |

---

## Fluxo Principal

| **Passo** | **Ações do Ator** | **Ações do Sistema** |
| :-------: | :---------------- | :------------------- |
| 1 | Acessa a opção "Atualizar Empresa". | |
| 2 | | Exibe os dados cadastrais e de contato atuais da empresa. |
| 3 | Atualiza os campos desejados e confirma. | |
| 4 | | Valida a Regra de Negócio **RN26** e a consistência dos dados obrigatórios. |
| 5 | | Salva as alterações e registra o histórico. |
| 6 | Visualiza a confirmação da atualização. | |

---
