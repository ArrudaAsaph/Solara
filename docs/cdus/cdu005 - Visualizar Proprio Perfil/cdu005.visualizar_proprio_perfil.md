# CDU005 – Visualizar Próprio Perfil

> **Arquivo:** `cdu005_visualizar_proprio_perfil.md`
> **Descrição:** Caso de uso referente à visualização dos próprios dados cadastrais pelo usuário autenticado. Nenhum dado é alterado neste caso de uso — para edição, ver CDU003-A.

---

## Identificação

| **Campo** | **Valor** |
| :-------- | :-------- |
| **Código** | CDU005 |
| **Ator Principal** | Usuário |
| **Atores Secundários** | Sistema |
| **Requisito Atendido** | RF16|



---

## Resumo

Permite que qualquer usuário autenticado e ativo visualize seus próprios dados cadastrais. O usuário vê todos os seus dados — incluindo os sensíveis (nome completo, CPF, tipo de perfil e status) — porém apenas em modo leitura. Para alterar dados não sensíveis, o usuário deve acessar o CDU003-A.


---

## Pré-condições

- O usuário deve estar autenticado no sistema.
- O usuário deve estar ativo.

---

## Pós-condições

- Os dados do próprio perfil são exibidos ao usuário.
- Nenhum dado é alterado no sistema.

---

## Mapeamento: CDU → Requisitos Funcionais

| **Requisito** | **Descrição** |
| :------------ | :------------ |
| RF16 | O usuário pode visualizar os seus próprios dados. |

---

## Mapeamento: CDU → Regras de Negócio

| **Código** | **Descrição** |
| :--------: | :------------ |
| **RN06** | Usuários desativados não podem autenticar nem acessar funcionalidades do sistema, incluindo a visualização do próprio perfil. |

---

## Fluxo Principal

| **Passo** | **Ações do Ator** | **Ações do Sistema** |
| :-------: | :---------------- | :------------------- |
| 1 | Acessa a opção "Meu Perfil". | |
| 2 | | Recupera os dados cadastrais do usuário autenticado. |
| 3 | | Exibe todos os dados do perfil em modo **somente leitura**: nome completo, CPF, tipo de perfil, status, e-mail, username, e-mail de contato e telefone. |
| 4 | Visualiza seus próprios dados. | |

---

## Fluxos de Exceção

### Fluxo de Exceção I – Usuário desativado tenta acessar o perfil

| **Passo** | **Ações do Ator** | **Ações do Sistema** |
| :-------: | :---------------- | :------------------- |
| 0.1 | Tenta acessar o sistema estando com status inativo. | |
| 0.2 | | Valida a Regra de Negócio **RN06**. |
| 0.3 | | Bloqueia o acesso e exibe mensagem informando que o usuário está inativo. |
